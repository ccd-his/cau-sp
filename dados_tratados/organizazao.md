# Pasta de Dados Tratados
Nesta pasta são colocados todos os dados tratados que serão utilizados para a construção do Indice de Resiliência Climática em situações de vulnerabilidade social, e seus subíndices.
Esses também são os dados disponíveis nos dashboards intermediários das entregas de produto para o CAU-SP.
Como a equipe não tem trabalhado diretamente com o github é possível notar data de alteração dos arquivos no próprio nome.
